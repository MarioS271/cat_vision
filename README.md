# CatVision

A minecraft mod which gives you Night Vision, even if you aren't a server operator. Inspired by CatEyes by Tee6! :)

With this mod, you can give yourself night vision with the click of a button.
The mod also has options to automatically enable night vision.
And the best part: you don't need any permission, just use it anywhere!
(And don't tell anyone, but you can even supress effects like Blindness or Nausea!)

**DISCLAIMER: I AM NOT LIABLE IF YOU GET BANNED FOR CHEATING USING THIS, PLEASE FOLLOW THE RULES OF WHATEVER
SERVER/REALM/MULTIPLAYER WORLD YOU ARE PLAYING ON WHEN USING THIS MOD!**

This is a rewrite of CatVision 1.4, now known as CatVision Legacy, in order to add compatibility for earlier versions like 1.16.5 or 1.20.1.
To view the old mod, visit [Modrinth](https://modrinth.com/mod/catvision-legacy) or [Curseforge](https://curseforge.com/minecraft/mc-mods/catvision).

### Current Features (v2.0):
- Smart Client-Side Night Vision
- Darkness/Blindness/Nausea Immunity
